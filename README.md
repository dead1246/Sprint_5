# Sprint_5

Автотесты на `pytest` + `selenium` для учебного сервиса Stellar Burgers.
Базовый URL проекта: `https://stellarburgers.education-services.ru`.

## Структура проекта

- `tests/` - UI-тесты, сгруппированные по функциональности.
- `conftest.py` - фикстуры и настройка браузера.
- `locators.py` - централизованные локаторы с комментариями.
- `helpers.py` - общие действия и ожидания для тестов.
- `data/credentials.py` - генераторы email и пароля.

## Что проверяется

- регистрация пользователя;
- ошибка при некорректном пароле;
- вход разными путями;
- переход в личный кабинет;
- возврат из личного кабинета в конструктор;
- выход из аккаунта;
- переключение вкладок конструктора.

## Запуск

1. Установить зависимости:

```bash
pip install -r requirements.txt
```

2. Запустить тесты в Chrome:

```bash
pytest -v
```

3. Запустить тесты в Firefox:

```bash
pytest -v --browser firefox
```

По умолчанию используется `chrome`. Для работы нужен установленный браузер.
